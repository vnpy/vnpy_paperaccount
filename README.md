## 说明

PaperAccount是用于本地仿真交易的功能模块，用户可以通过其UI界面基于实盘行情进行本地化的模拟交易。

## 安装

安装环境推荐基于4.0.0版本以上的【[**VeighNa Studio**](https://www.vnpy.com)】。

直接使用pip命令：

```
pip install vnpy_paperaccount
```


或者下载源代码后，解压后在cmd中运行：

```
pip install .
```
